iOS to tvOS
=========

## 👍 Introduction

This is a basic sample showing how to communicate from an iOS app to a tvOS app.

## 🌟 Features

- [x] Automatic connection from the iOS client to the tvOS server

## ⚠️ Requirements

* xCode 7.1 beta
